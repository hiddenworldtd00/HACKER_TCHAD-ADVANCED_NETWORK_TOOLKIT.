Voici une description détaillée des 13 fonctionnalités intégrées à notre toolkit :

1. Test Ping : Utilise les requêtes ICMP pour vérifier la disponibilité d'un hôte et mesurer la latence réseau.
2. Traceroute : Permet de visualiser le chemin complet que prennent les paquets pour atteindre la destination, en identifiant chaque saut (routeur) intermédiaire.
3. DNS Lookup (A) : Résout un nom de domaine en son adresse IP associée pour localiser la cible.
4. MX/NS/TXT Records : Extrait les configurations de messagerie, les serveurs de noms et les enregistrements textuels liés à un domaine.
5. Reverse DNS : Effectue une recherche inverse pour identifier le nom d'hôte correspondant à une adresse IP spécifique.
6. Whois Lookup : Récupère les données d'enregistrement d'un nom de domaine, incluant les informations sur le propriétaire et les dates d'expiration.
7. IP Geolocation : Détermine la position géographique approximative d'une adresse IP.
8. ASN Lookup : Identifie le système autonome (ASN) qui gère le bloc d'adresses IP pour comprendre l'infrastructure réseau de l'hébergeur.
9. TCP Port Scan : Analyse les ports ouverts sur une machine cible pour identifier les services actifs et potentiellement vulnérables.
10. HTTP Headers : Analyse les en-têtes de réponse d'un serveur web pour révéler des informations sur la technologie utilisée et la configuration du serveur.
11. Extract Links : Récupère toutes les URL présentes dans une page web pour cartographier les ressources liées.
12. Analytics Search : Identifie les codes de suivi analytique sur un site pour corréler plusieurs propriétés web entre elles.
13. Zone Transfer (AXFR) : Tente de copier l'intégralité de la zone DNS d'un serveur pour obtenir une vision complète de l'architecture sous-domaine d'une cible.
